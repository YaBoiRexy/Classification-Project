# Classification Project

I trained a machine learning model on the Kaggle Credit Dataset, predicting credit card defaults with 85% accuracy. See [my portfolio](https://ubc.craft.me/dana/b/7C662A59-7FBE-44D4-BE09-5E1870B69D8C/End-to-End-Machine-Learning-Pipeline) for extended summary.

## How to view

1. Open "Classification Project.ipynb" on GitHub.

## How to run

1. Clone repo.
2. Use Jupyter Lab to open "Classification Project.ipynb".
3. Click "Restart kernel and run all cells" <img width="20" alt="Restart button" src="https://github.com/user-attachments/assets/72b37e2c-3db0-444c-b60c-a8348e864c2e" />.
4. Wait 10-20 minutes for model training.
